# tfk-form-skoleskyss
Form plug-in for skoleskyss

Based on Google Web Starter Kit

# Setup

```
$ npm run setup
```

# Develop

```
$ npm run dev
```

# Build

```
$ npm run build
```

# Start

```
$ npm run start
```

See form at

[http://localhost:3000/skoleskyss](http://localhost:3000/skoleskyss)
